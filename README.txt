***********************************
|       Simple Ads JSON           |
***********************************
Drupal 7 module.

Return Information on Ads in JSON format.

Requirements:

SimpleAds 7.x

Instruction:
Request url format:
example.com/simpleads/json?start=Y-m-d
example.com/simpleads/json?start=Y-m-d&end=Y-m-d

For Example:
To request information related to ads with starting date on 2016-1-1
example.com/simpleads/json?start=2016-1-1

and with ending date on 2016-3-1
example.com/simpleas/json?start=2016-1-1&end=2016-3-1
